## 16 型人格详细旅行指南

### 分析师组（NT）

#### INTJ 建筑师
```
🎯 旅行风格：深度探索型

✅ 适合：
- 历史文化深度游（博物馆、古迹、遗址）
- 独自旅行或小团
- 有明确目标的主题旅行
- 建筑、艺术鉴赏之旅
- 冷门但有内涵的目的地

❌ 避免：
- 走马观花的跟团游
- 人山人海的网红景点
- 没有规划的随便走走
- 纯购物行程

🏨 住宿：安静、私密性好、有设计感
🍜 餐饮：不需要网红店，追求品质和安静
⏰ 节奏：紧凑但深度，每个景点充分了解
👥 同行：独行或志同道合的1-2人

📍 推荐目的地类型：
- 历史古城（西安、罗马、雅典）
- 文化之都（京都、维也纳）
- 自然+人文（新疆、西藏）
```

#### INTP 逻辑学家
```
🎯 旅行风格：好奇探索型

✅ 适合：
- 小众冷门目的地
- 科技馆、天文馆、博物馆
- 自由行、无固定行程
- 探索城市的隐藏角落
- 长时间泡在一个感兴趣的地方

❌ 避免：
- 分秒必争的行程
- 社交型团队游
- 千篇一律的打卡点
- 被催促、被安排

🏨 住宿：安静、有特色、WiFi要好
🍜 餐饮：可以为了一家有意思的店专门去
⏰ 节奏：松散灵活，允许临时改变
👥 同行：独行或能互相给空间的朋友

📍 推荐目的地类型：
- 科技城市（东京、深圳、硅谷）
- 有故事的小镇（布拉格、爱丁堡）
- 自然奇观（冰岛、新西兰）
```

#### ENTJ 指挥官
```
🎯 旅行风格：征服成就型

✅ 适合：
- 地标打卡、名胜古迹
- 高效紧凑的行程
- 有挑战性的体验（登山、极限运动）
- 商务+旅行结合
- 带队组织团队旅行

❌ 避免：
- 无目标的闲逛
- 浪费时间排队
- 太过休闲的度假
- 低效的行程安排

🏨 住宿：高档、位置好、效率高
🍜 餐饮：有名气、值得打卡
⏰ 节奏：紧凑充实，最大化时间价值
👥 同行：可以带队，或与同样高效的人

📍 推荐目的地类型：
- 国际大都市（纽约、伦敦、东京）
- 世界奇迹（长城、金字塔）
- 挑战型目的地（珠峰大本营、撒哈拉）
```

#### ENTP 辩论家
```
🎯 旅行风格：新奇体验型

✅ 适合：
- 新奇独特的体验
- 多元文化碰撞的地方
- 随时可能改变的行程
- 和有趣的人交流
- 尝试当地新鲜事物

❌ 避免：
- 一成不变的常规路线
- 太过安静无聊的地方
- 完全按计划走
- 重复去同一个地方

🏨 住宿：特色民宿、有故事的地方
🍜 餐饮：猎奇、当地特色、敢于尝试
⏰ 节奏：灵活多变，保持新鲜感
👥 同行：有趣的朋友、能聊得来的人

📍 推荐目的地类型：
- 多元文化城市（香港、新加坡、洛杉矶）
- 有冲突感的地方（柏林、耶路撒冷）
- 冒险目的地（摩洛哥、古巴）
```

## 行程定制应用

当用户告知 MBTI 后，在行程规划中体现：

1. **行程节奏**：I人松散 / E人丰富
2. **景点选择**：N人深度 / S人体验
3. **住宿风格**：T人实用 / F人氛围
4. **社交安排**：I人独处时间 / E人交流机会
5. **计划弹性**：J人详细计划 / P人灵活空间

### 输出示例
```
💡 根据你的 MBTI（INFP），我为你定制了这份行程：

✨ 行程特点：
- 节奏舒缓，不赶时间
- 侧重自然风光和文艺氛围
- 预留独处和发呆的时间
- 选择有设计感的住宿
- 避开人多嘈杂的景点

📍 特别推荐：
- [符合INFP的景点]
- [适合INFP的咖啡馆]
- [INFP会喜欢的体验]
```
